1. First of all, choose an integer <n>.

2. `cd public-testcase/<n>; mkdir dir<n>; mkdir -p output/files`

3. Generate files
`../../generate-random-fs.sh dir<n> 10 1000 20`

4. Generate correct Merkle tree files and proofs
`../../merkle-dir.sh build dir<n> --output output/files/dir<n>.mktree`
`for i in $(find dir<n> -type f -printf '%P '); do ../../merkle-dir.sh gen-proof "$i" --tree output/files/dir<n>.mktree --output "output/files/$(basename $i).proof"; done`

5. Create a file `input`, which contains:
```
# TEST GROUP: Problem 1
<custom argument for each command to be tested here>

# TEST GROUP: Problem 2
build dir<n> --output your_output/dir<n>.mktree
build --output your_output/dir<n>.mktree dir<n>

# TEST GROUP: Problem 3
<p3>

# TEST GROUP: Problem 4
<p4>
```

+ `<p3>` is generated by the command:
    `for i in $(find dir<n> -type f -printf '%P\n'|LC_COLLATE=C sort|tr '\n' ' '); do echo "gen-proof $i --tree output/files/dir<n>.mktree --output your_output/$(basename $i).proof"; done`
+ `<p4>` is generated by the command:
    `for i in $(find dir<n> -type f -printf '%p\n'|LC_COLLATE=C sort|tr '\n' ' '); do echo "verify-proof $i --proof output/files/$(basename $i).proof --root <root_hash>"; done`
    where `<root_hash>` is the last line of the file `output/files/dir<n>.mktree`.
